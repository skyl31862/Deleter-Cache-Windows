README – Cache Windows Cleaner (.bat)

============================ FRANÇAIS ============================

Ce script Batch (.bat) permet de nettoyer rapidement
certains fichiers temporaires de Windows afin de libérer de l’espace et
améliorer les performances.

Il peut supprimer : - le cache utilisateur Windows - le cache système
Windows - le dossier Temp - le dossier Prefetch

Comment utiliser le script

1.  Place le fichier .bat sur ton ordinateur.
2.  Clique droit sur le fichier.
3.  Clique sur “Exécuter en tant qu’administrateur” (recommandé).
4.  Une fenêtre de commande s’ouvre et pose quelques questions.

Nettoyage du cache utilisateur

Le script demande : Voulez vous supprimer les caches d’utilisateur
Windows [O] Oui [N] Non

Nettoyage du cache Windows

Le script demande : Voulez vous supprimer les caches Windows [O] Oui [N] Non

Important

Certains fichiers peuvent ne pas être supprimés s’ils sont utilisés par
Windows. Il est conseillé de fermer les programmes avant d’exécuter le
script.

============================ ENGLISH ============================

This Batch (.bat) script quickly cleans Windows temporary
files to free disk space and improve performance.

It can delete: - Windows user cache - Windows system cache - Temp
folder - Prefetch folder

How to use

1.  Place the .bat file on your computer.
2.  Right-click the file.
3.  Click “Run as administrator” (recommended).
4.  A command window will open and ask questions.

User cache cleaning

The script will ask: Do you want to delete Windows user cache [O] Yes [N] No

Windows cache cleaning

The script will ask: Do you want to delete Windows cache [O] Yes [N] No

Important

Some files may not be deleted if they are currently in use. It is
recommended to close running programs before executing the script.
